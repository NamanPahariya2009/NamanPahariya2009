### 🛠️ Featured Project: [NEXUS](link-to-repo)
> **Status:** Active Development | Version: 5.0 (HUD Redesign)
Nexus is a unified AI ecosystem designed to bridge the gap between laptop and mobile environments.
* **Intelligence:** Hybrid logic using **Ollama (Gemma2:2b)** for local edge processing and **Gemini 2.5 Flash** for cloud-based vision tasks.
* **Infrastructure:** Custom **ADB Wireless Bridge** for seamless Android control (Calls, WhatsApp, Media).
* **Interface:** Ultra-premium HUD built with **Tkinter** and **OpenCV** surveillance feeds.

### 🔌 Hardware & Engineering

* **IoT Ecosystem:** ESP32/Arduino connectivity with custom PCB architecture.
* **3D Design:** Mechanical enclosures designed for integrated electronics.
